* 64 bit version only

	* .NET 4.6.2
	* CRT 2017 - https://go.microsoft.com/fwlink/?LinkId=746572
* Remaining:
    * Vehicle sample and a few related classes
    * Attach/Detach shape from an actor (recently added, but needs testing)
    * Getting started guide for people
    * Serialization
    * Broad Phase
    * Deletion Listener
